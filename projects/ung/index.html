Universal Name Generator
Tool

About

Tool that can generate accurate new words and names in any given language, be it real or fictional.


Markov

A generator featuring Markov chains using Katz's back-off model, for word generation using the same gramatical constructs as those of the desired language

Mixer

A simple and random mixer generator, combining multiple given bits into new words or names
