Minesweeper

Minesweeper is a very similar game to the original game with the same name. It was created to fill the gap after the original's removal from modern Windows releases.

The objective is to carefully clear all the mines on a rectangular board, without detonating them.

  • Language: C#
  • Technologies: .NET, GTK2
  • Operating Systems: Windows, Mac, Linux
